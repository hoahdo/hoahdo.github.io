Trivia App

Technologies: HTML, CSS, JavaScript, React, Netlify

This project was inspired by a challenge on the Scrimba React Course. It is built with React and connects to the Open Trivia Database API. I expanded upon the original challenge by implementing custom setting selectors so users can choose their desired category, difficulty, and number of questions. I also added replay options for users to play again with the same settings or return to the main page to select new settings. It is hosted via Netlify.

Event Planner

Technologies: HTML, CSS, JavaScript, React, Express JS, Node JS, MongoDB, Tailwind CSS, React Router, Netlify, Render

I built this project to help organize and manage events. It is built with the MERN stack and uses the React Router library to enable page routing. The events are saved in a MongoDB database in the cloud so that they can be viewed from anywhere. I utilized the unopinionated framework, Tailwind CSS, to style the app. Users can view all the event posts that have been created on a single page. From there, they can create a new event post, update a current event post, or delete a current event post.
